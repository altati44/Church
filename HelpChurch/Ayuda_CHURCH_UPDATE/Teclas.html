Teclas de Acceso Rápido

A continuación le mostramos las teclas de acceso rápido que funcionan en nuestra aplicación, todas ellas están reflejadas junto al título de cada acceso, pero aquí las tiene todas por si las necesita.

 

Alt+X---------------> Salir de la Aplicación.

Ctrl+A--------------> Abrir Base de Datos.

Ctrl+C--------------> Copiar.

Ctrl+V--------------> Pegar.

Ctrl+P--------------> Imprimir.

Ctrl+P--------------> Cortar.

F1------------------> Mostrar ayuda.

F2------------------> Buscar Tabla Maestra.

F5------------------> Abrir ventana Importar.

F6------------------> Abrir ventana Exportar.

F7------------------> Cerrar Base de Datos Externa.

F8------------------> Eliminar registros sueltos.

F9------------------> Configurar Tabla Maestra.

F10-----------------> Configurar Página.

Del/Delete/Suprimir-> En la rejilla de datos se utiliza para borrar registros, en el Panel de Fotografía, para borrar la foto, en otros lugares para el borrado del texto.

Enter/Return--------> En la rejilla de datos para pasar a la siguiente cuadrícula, en la mayoría de las ventanas para aceptar los cambios.

Esc/Escape----------> En la rejilla de datos aborta los cambios, en la mayoría de las ventanas sale e ignora los cambios.
